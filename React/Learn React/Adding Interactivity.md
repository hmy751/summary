## 이벤트 핸들러 

이벤트 핸들러에 전달되는 함수는 호출이 아니라 전달되어야 한다.
왜냐 하면 JSX의 중괄호 내부는 바로 실행이 되는데 호출로 전달하면 렌더링 중에 즉시 함수를 실행하기 때문이다.
인라인으로 전달할때는 익명 함수로 화살표 함수를 통해서 전달할 수 있다.
